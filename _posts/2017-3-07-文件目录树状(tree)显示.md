#### 显示在右侧目录树上
Tab退格 插入 **代码** 

	$ brew install tree

#### 使用
- 小黑点 'bullet point' 直接使用 `tree` 命令，会在当前文件目录下，递归输出所有文件层级
	
		$ tree

- 限制层级

		$ tree -L 2

- 指定当前目录下的某个文件夹

		$ tree Desktop
	
#### 导出文件  
用`> 文件名.格式` 的形式导出

	$ tree -L 1 > tree.md
